Gashabits
A website where you can turn 15¢ of Litecoin ($LTC) into unique, generated pixel art.
Your Litecoin transaction ID (TXID) acts as the exact mathematical seed to build the art. Every single pull guarantees a unique digital asset with no lose states.
Features

    Hash-Generated Art: The blockchain transaction hash directly dictates the visual style and patterns of the pixel art.
    Low Fees: Uses the Litecoin network to keep transaction fees under a penny.
    Bulk Pulls: Buy multiple pulls at once and download all your artwork instantly as a ZIP file.
    No Downloads: Works entirely in your web browser with no wallet extensions required.

How It Works

    Select how many pixel arts you want to generate (15¢ per pull).
    Send the exact amount of Litecoin to the displayed address.
    The backend verifies the transaction and instantly generates your downloadable art.
